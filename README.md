# Smart File Organizer 

A Python script that automatically organizes files in your Downloads folder by type.

## Features 
- Watches a folder in real-time
- Sorts files into categories (Images, Documents, Videos, etc.)
- Handles duplicate filenames
- Clean terminal output using Rich

## Setup 
'''bash 
python3 -m venv venv
source venv/bin/activate 
pip install -r requirements.txt 
python main.py
'''bash
